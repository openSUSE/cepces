# Protocol References

This document provides references to the Microsoft protocol specifications
implemented by cepces.

## MS-XCEP: X.509 Certificate Enrollment Policy Protocol

The X.509 Certificate Enrollment Policy Protocol (MS-XCEP) is used to retrieve
certificate enrollment policies from an Active Directory Certificate Services
(AD CS) server.

- **Specification**: [MS-XCEP](https://learn.microsoft.com/en-us/openspecs/windows_protocols/ms-xcep/08ec4475-32c2-457d-8c27-5a176660a210)
- **Overview**: [MS-XCEP Overview](https://learn.microsoft.com/en-us/openspecs/windows_protocols/ms-xcep/3b85f15b-a682-424a-9db6-a39f5613066b)

## MS-WSTEP: WS-Trust X.509v3 Token Enrollment Extensions

The WS-Trust X.509v3 Token Enrollment Extensions (MS-WSTEP) protocol is used
to request and receive X.509 certificates from an AD CS server.

- **Specification**: [MS-WSTEP](https://learn.microsoft.com/en-us/openspecs/windows_protocols/ms-wstep/4766a85d-0d18-4fa1-a51f-e5cb98b752ea)

## Related Protocols

### MS-CERSOD: Certificate Services Protocols Overview

Provides an overview of how the certificate enrollment protocols work together.

- **Specification**: [MS-CERSOD](https://learn.microsoft.com/en-us/openspecs/windows_protocols/ms-cersod/8673f247-e97d-4a29-8c35-0e4eeac9e006)

### WS-Trust

The underlying web services trust specification used by MS-WSTEP.

- **Specification**: [WS-Trust 1.3](http://docs.oasis-open.org/ws-sx/ws-trust/200512/ws-trust-1.3-os.html)
